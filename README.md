# Personal Website

A minimal personal website built with Jekyll, inspired by stefanyas.me.

## Setup

1. Make sure you have Ruby installed on your system
2. Install Jekyll and dependencies:
   ```bash
   gem install bundler
   bundle install
   ```

## Running Locally

To run the website locally:

```bash
bundle exec jekyll serve
```

Then open your browser and navigate to `http://localhost:4000`

## Customization

### Personal Information

#### Home Page (`index.html`)
- Add your profile photo to `assets/images/profile.jpg` (200x200px recommended)
- Update the profile summary with your role, interests, and skills
- Update social media links (Instagram, Email, LinkedIn, Codecademy, GitHub)

#### About Page (`about.html`)
- Fill in your background and career journey
- Add your education and qualifications
- List your skills and expertise
- Describe your activities and interests

#### Projects Page (`projects.html`)
- Add project screenshots to `assets/images/project1.jpg` and `assets/images/project2.jpg`
- Update project titles and descriptions
- List technologies used for each project
- Add key features for each project
- Update GitHub and live demo links

### Site Configuration

Edit `_config.yml` to update:
- Site title
- Description
- Other Jekyll settings

### Styling

The main stylesheet is located at `assets/css/style.css`. You can customize:
- Colors
- Fonts
- Spacing
- Layout

### Image

Replace `assets/images/ellipse.svg` with your own image if desired.

## Structure

```
personal-website/
├── _config.yml          # Jekyll configuration
├── _layouts/
│   └── default.html     # Main layout template
├── assets/
│   ├── css/
│   │   └── style.css    # Stylesheet
│   └── images/
│       └── ellipse.svg  # Ellipse image
├── index.html           # Homepage
├── Gemfile              # Ruby dependencies
└── README.md            # This file
```

## Deployment

You can deploy this website to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

For GitHub Pages, simply push to a repository and enable GitHub Pages in the repository settings.

## License

Feel free to use this template for your own personal website.
