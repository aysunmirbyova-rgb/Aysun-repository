# Image Assets

Please add your images to this folder:

## Required Images:

1. **profile.jpg** - Your profile photo
   - Recommended size: 200x200px or larger (square)
   - Format: JPG, PNG, or WebP
   - This appears on the home page

2. **project1.jpg** - Screenshot of your first project
   - Recommended size: 800x600px or larger
   - Format: JPG or PNG
   - This appears on the projects page

3. **project2.jpg** - Screenshot of your second project
   - Recommended size: 800x600px or larger
   - Format: JPG or PNG
   - This appears on the projects page

## How to Add Images:

1. Replace the placeholder files with your actual images
2. Keep the same filenames (profile.jpg, project1.jpg, project2.jpg)
3. Or update the image paths in the HTML files if you want to use different names

The website will automatically display your images once you add them to this folder!
